## 🔒 セキュリティ重要メモ

### APIキーがGitHubに公開されている件について
**これは問題ない。設計通り。** 以下の理由で安全：

1. GitHub Pagesは静的サイトなので、APIキーは必ずブラウザに露出する
   （ブラウザの開発者ツールで誰でも見れる前提）
2. クライアントサイドJSでは環境変数が使えない
3. 正しい守り方は「漏れないようにする」ではなく「**漏れても悪用されない制限**」
4. Maps Platform API KeyにはHTTPリファラー制限（`tkkikaku.github.io/*`）が設定済み
5. 他のドメインでは使えないため、キーを知られても悪用できない

### 新しいAPIキーを発行する場合の必須設定
**必ず以下の制限を設定すること**：
- アプリケーションの制限：ウェブサイト
- 許可サイト：`tkkikaku.github.io/*`（または使用ドメイン）
- API制限：必要最小限のAPIのみ許可

制限なしキーをコミットすると請求額爆発リスクあり。

### 過去の対応履歴
- 2026年以前：APIキー漏洩について一度対応済み（リファラー制限を設定）
